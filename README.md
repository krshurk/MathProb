MathProb — это образовательная веб-платформа, разработанная для помощи старшеклассникам в изучении теории вероятностей. В отличие от статических учебников, наш проект предлагает интерактивные тесты, структурированные учебные материалы и отслеживание прогресса для повышения вовлеченности и успеваемости.

Проект создается с использованием ASP.NET Core и нацелен на предоставление удобного инструмента как для самостоятельного обучения, так и для дополнительной подготовки к контрольным работам и экзаменам
